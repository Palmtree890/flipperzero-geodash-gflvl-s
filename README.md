# flipperzero geometry dash gflvl level files
this is a collection of levels for the in development geometry dash game for the flipper zero created by gooseprjkt
## how to use
put the game files in SD card/apps_data/geometry_flip
I recomend using a application such as [qflipper](https://flipper.net/pages/downloads) for the file transfer

This game was created by the user [gooseprjkt](https://github.com/gooseprjkt/)
this is simply a collection of level files for his [game](https://github.com/gooseprjkt/flipperzero-geometryflip)

### notice
I took no part in the development of this game. all of the app was created by gooseprjkt and I just created levels for it.
I am in no way related to the project or at all a contributer. I simply just created unofficial levels for it.
